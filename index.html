¿Cómo usar este dashboard?

  1. Sube cualquier archivo CSV - El sistema detectará automáticamente los encabezados
  2. Configura el presupuesto - Ajusta el monto total y el monto individual según tus datos reales
  3. El sistema identifica automáticamente quiénes recibieron el reembolso (columna REEMBOLSOS PAGADOS)
  4. Los cálculos se actualizarán automáticamente basados en las personas que realmente recibieron el pago
  5. Usa los filtros para ver solo pagados, no pagados, o todos los registros
